Basic structs of HTML
What is HTML
HTML(.html files) is useful at network,if you opened a page like this,
you should know this page is build on HTML.HTML is not a program
language,it is a hyper text,but it is look like scripts.If you want to create
your HTML file,first you need a editor like Microsoft Visual Studio Code,
Or......Dev-C++ or notepad?It's OK,but I think Microsoft VSCode is best.
Your first HTML file
Create a HTML file and fill it like:
1|<html>
2|<head>
3|<title>HTML File</title>
4|</head>
5|<body>
6|Hello World
7|<!--这是注释,上面的是文本-->
8|</body>
9|</html>


















Yes,all items is at <html> to </html>
Some setup configs is at "head",more item is at "body".
In html->body,type some text equal output in C++.
Something in <!-- --> is equal // in C++ or # in python.
<br> equal print '\n' in C++,<hr> can draw a line.
